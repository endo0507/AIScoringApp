## 使い方
1. 「ファイルを選択」(Choose File)を押し、画像を開く。
2. 「問題・採点条件」のところにある、「認識」を押し、問題文と採点条件が書かれているところを画像内から範囲選択する。
3. 同様に、「解答」についても2. と同じ操作を行う。
4. 「問題・採点条件」と「解答」の文字認識が正しく行えていたら、「採点開始」を押して、AIによる採点を行う。
5. 「採点結果」欄の内容を確認する。
## 既知の不具合
- 「問題・採点条件」および「解答」の文字認識完了まで30秒程度以上かかる
- 認識精度が低いかも（誤認識した場合は、手入力で修正可能です）

現時点で判明しているこれらの不具合については、将来的に改善・解決する予定です。
